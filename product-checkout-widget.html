Product Application

Checkout Widget

Embedded widget for merchant checkout pages. Handles point earning and redemption during purchase.

Theming & Embedding

The checkout widget is embeddable and must support theming for various host contexts including Shopify Polaris, custom merchant sites, and platform integrations.

CSS Isolation Strategy

The widget must be isolated from host page styles to prevent conflicts.

Shadow DOM

Full style encapsulation. Styles cannot leak in or out.

Recommended

Tailwind Prefix

All classes prefixed with sw- to avoid collisions.

Alternative

iframe Embed

Complete isolation via sandboxed iframe. Cross-origin communication via postMessage.

Legacy

// Shadow DOM initialization

const host = document.querySelector('#saltwyk-widget');
const shadow = host.attachShadow({ mode: 'open' });
shadow.innerHTML = `
  <style>${widgetStyles}</style>
  <div class="sw-checkout-widget">...</div>
`;

  }
});
