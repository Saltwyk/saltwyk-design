Core

Design tokens, brand assets, and UI components shared across all product surfaces

Core defines the building blocks: color scales, type families, spacing units, motion curves, brand assets, and UI components. These elements are shared across every surface—Shopper, Merchant, Marketing, and Docs.

What differs between surfaces is composition—how these elements combine. Fire Opal and Bright Opal treatments use the same thread colors at different saturation levels. Core provides the palette; product surfaces determine the application.
