Core Foundation

Gradient System

Gradients serve specific purposes in the Saltwyk design system. Security thread gradients create the modern currency aesthetic. Brand and surface gradients provide depth and visual interest for heroes, backgrounds, and compositions.

Security Threads

Thread Gradients

High-saturation fluorescent gradients inspired by security threads in modern currency. Use for thread lines (1-3px), text treatments, and brand moments. These gradients are attention-grabbing by design—use sparingly.

Compositions

Thread + Tint Hero Pattern

Full-power thread lines live next to tinted backgrounds—never as fills. Thread lines at the top of containers should be full-bleed with squared top corners.

Corner treatment: When placing a thread at the top of a container, use squared top corners (rounded-b-lg) to allow the thread to extend full width. Rounded top corners create an awkward gap between the thread and container edge.
