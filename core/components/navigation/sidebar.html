Design Decision: Navigation active states use indigo (selection color) rather than emerald (action/success color). This maintains consistency with other selection patterns like tabs, table rows, and form controls. Emerald is reserved for action buttons and positive/success semantics like the "New" badge.

9. Code Reference

Sidebar Container

.sidebar { width: 240px; background: white; border-right: 1px solid hsl(var(--warm-200)); display: flex; flex-direction: column; } .sidebar-dark { background: hsl(var(--intaglio-900)); border-right: none; }

Logo Area

.sidebar-logo { height: 64px; padding: 0 20px; display: flex; align-items: center; gap: 10px; border-bottom: 1px solid hsl(var(--warm-200)); }

Nav Item Styles

.sidebar-item { display: flex; align-items: center; gap: 12px; padding: 10px 16px; margin: 0 8px; border-radius: 6px; font-size: 14px; font-weight: 500; color: hsl(var(--warm-600)); cursor: pointer; transition: all 0.15s ease; } /* Active state - uses INDIGO */ .sidebar-item-active { background: hsl(var(--indigo-50)); color: hsl(var(--indigo-700)); border-left: 3px solid hsl(var(--indigo-500)); margin-left: 5px; padding-left: 13px; } /* Dark mode active */ .sidebar-dark .sidebar-item-active { background: hsl(var(--indigo-500) / 15%); color: hsl(var(--indigo-300)); border-left-color: hsl(var(--indigo-400)); }

10. Implementation Checklist

Light Mode

  • Container: 240px width, white background, warm-200 border-right
  • Logo area: 64px height, 20px horizontal padding
  • Nav item: 10px 16px padding, 6px radius, warm-600 text
  • Active state: indigo-50 bg, indigo-700 text, indigo-500 left border (3px)
  • Focus ring: 2px white gap, 4px indigo-500 ring
  • Badge: magenta-500 background, white text

Dark Mode (Fire Opal)

  • Container: 240px width, intaglio-900 background, no border
  • Nav item: warm-400 default text, warm-500 icon
  • Hover: intaglio-800 bg, warm-200 text
  • Active state: indigo-500/15% bg, indigo-300 text, indigo-400 left border
  • Focus ring: 2px intaglio-900 gap, 4px indigo-400 ring
  • Badge: magenta-400 background

Collapsed State

  • Width: 64px
  • Hide: labels, badges, section headers
  • Center: icons and avatar
  • Tooltip: right of item on hover, ink background
  • Animation: 200ms ease-out width transition

Nested Navigation

  • Parent: chevron icon rotates 90deg when expanded
  • Children: 24px left indent, 13px font size
  • Parent highlighted when any child is active
  • Animation: 150ms fade + slide down
