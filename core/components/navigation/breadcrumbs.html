4. Truncation Behavior

Handling long breadcrumb paths with ellipsis overflow.

Truncated Example

Home / / Settings / Notifications

Truncation Guidelines:

  • Use ellipsis (...) when paths exceed 4-5 levels
  • Ellipsis should be clickable to reveal a dropdown with hidden levels
  • Always show the first (root) and last 2-3 items
  • Include a subtle chevron-down indicator on the ellipsis button

5. Code Reference

CSS implementation for breadcrumb styling.

/* Container */ .breadcrumbs { @apply flex items-center gap-2 text-sm; } /* Clickable links */ .breadcrumb-link { @apply text-warm-600 hover:text-ink hover:underline; @apply transition-colors duration-150; } /* Separator (/ or chevron) */ .breadcrumb-separator { @apply text-warm-400; } /* Current page (not clickable) */ .breadcrumb-current { @apply text-ink font-medium; } /* Dark mode overrides */ .breadcrumbs-dark .breadcrumb-link { @apply text-warm-400 hover:text-warm-200; } .breadcrumbs-dark .breadcrumb-separator { @apply text-warm-600; } .breadcrumbs-dark .breadcrumb-current { @apply text-warm-100; }

6. Implementation Checklist

Key specifications to verify for breadcrumb implementation.

Visual

  • ☐ Gap between items: 8px
  • ☐ Font size: 14px (text-sm)
  • ☐ Link color: warm-600
  • ☐ Current item: font-medium (500)
  • ☐ Separator color: warm-400

Interaction & Accessibility

  • ☐ Links show underline on hover
  • ☐ Current page is not a link
  • ☐ Use <nav> element with aria-label="Breadcrumb"
  • ☐ Current item uses aria-current="page"
  • ☐ Separators hidden from screen readers
