5. Implementation Checklist

Key specifications to verify for select menus.

Trigger

  • ☐ Border radius: 6px
  • ☐ Border: 1.5px warm-300
  • ☐ Padding: 10px 12px
  • ☐ Focus ring: indigo-500/20
  • ☐ Error border: magenta-500

Menu

  • ☐ Width matches trigger
  • ☐ Selected item shows checkmark
  • ☐ ARIA: role="listbox"
  • ☐ Items: role="option"
