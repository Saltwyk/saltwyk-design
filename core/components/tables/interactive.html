5. Code Reference

CSS implementation for interactive table features.

/* Sortable Column Headers */ .sortable { cursor: pointer; user-select: none; } .sortable:hover { color: hsl(24 29% 8%); } .sort-icon { opacity: 0.4; margin-left: 4px; } .sortable:hover .sort-icon, .sortable.active .sort-icon { opacity: 1; } /* Checkbox Styling */ .table-checkbox { width: 18px; height: 18px; border: 2px solid hsl(30 5% 78%); border-radius: 4px; appearance: none; } .table-checkbox:checked { background: hsl(230 28% 42%); /* indigo-500 */ border-color: hsl(230 28% 42%); } /* Selected Row State */ .row-selected { background: hsl(230 25% 96%); /* indigo-50 */ border-left: 3px solid hsl(230 28% 42%); } .row-selected-dark { background: hsl(230 28% 42% / 10%); border-left: 3px solid hsl(230 26% 54%); } /* Bulk Action Bar */ .bulk-action-bar { display: flex; justify-content: space-between; padding: 12px 16px; background: hsl(230 25% 96%); border: 1px solid hsl(230 32% 82%); border-radius: 8px; } /* Expand Button */ .expand-btn { width: 24px; height: 24px; border-radius: 4px; transition: all 0.15s ease; } .expand-btn.expanded svg { transform: rotate(90deg); }

6. Implementation Checklist

Key specifications to verify for interactive table features.

Sortable Columns

  • ☐ Cursor: pointer on sortable headers
  • ☐ Icon opacity 0.4 default, 1 on hover/active
  • ☐ ARIA: aria-sort attribute
  • ☐ Correct icon for each state

Selectable Rows

  • ☐ Checkbox: 18px, 2px border, 4px radius
  • ☐ Checked: indigo-500 background
  • ☐ Row selected: indigo-50 bg, 3px border
  • ☐ Indeterminate state for header

Bulk Actions

  • ☐ Appears when rows selected
  • ☐ Shows selection count
  • ☐ indigo-50 background (light)
  • ☐ indigo-500/15% background (dark)

Expandable Rows

  • ☐ Expand button: 24px, chevron-right
  • ☐ 90deg rotation when expanded
  • ☐ Expanded: warm-50 (light), intaglio-900 (dark)
  • ☐ Border-top separates content
