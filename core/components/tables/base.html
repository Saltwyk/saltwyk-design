5. Code Reference

CSS classes for table container and base styles.

Table Container

/* Light mode container */
.table-container {
  background: white;
  border: 1px solid hsl(30 5% 88%);
  border-radius: 8px;
  overflow: hidden;
}

/* Dark mode container */
.table-container-dark {
  background: hsl(24 28% 12%);
  border: 1px solid hsl(24 27% 20%);
  border-radius: 8px;
  overflow: hidden;
}

Base Table

.table-base {
  width: 100%;
  border-collapse: collapse;
}

.table-base th {
  text-align: left;
  font-size: 12px;
  font-weight: 600;
  text-transform: uppercase;
  letter-spacing: 0.05em;
  padding: 12px;
}

.table-base td {
  padding: 12px;
  font-size: 14px;
}

Light Mode Colors

.table-light th {
  background: hsl(30 5% 97%);
  color: hsl(30 6% 46%);
  border-bottom: 1px solid hsl(30 5% 88%);
}

.table-light td {
  color: hsl(24 29% 8%);
  border-bottom: 1px solid hsl(30 5% 95%);
}

.table-light tbody tr:hover {
  background: hsl(30 5% 97%);
}

Variant Classes

/* Striped */
.table-striped tbody tr:nth-child(even) {
  background: hsl(30 5% 97%);
}

/* Bordered */
.table-bordered th,
.table-bordered td {
  border: 1px solid hsl(30 5% 88%);
}

/* Density */
.table-compact th, .table-compact td {
  padding: 8px 12px;
}
.table-comfortable th, .table-comfortable td {
  padding: 16px;
}

6. Implementation Checklist

Verify these specifications when implementing base tables.

Container

  • ☐ Border radius: 8px
  • ☐ Border: 1px solid warm-200 (light) / intaglio-700 (dark)
  • ☐ Overflow: hidden (for rounded corners)
  • ☐ Background: white (light) / intaglio-800 (dark)

Table Header

  • ☐ Font size: 12px
  • ☐ Font weight: 600
  • ☐ Text transform: uppercase
  • ☐ Letter spacing: 0.05em
  • ☐ Background: warm-50 (light) / intaglio-900 (dark)

Table Body

  • ☐ Font size: 14px
  • ☐ Cell padding: 12px (default)
  • ☐ Row border: 1px solid warm-100 (light)
  • ☐ Last row: no bottom border
  • ☐ Hover state: warm-50 background

Variants

  • ☐ Striped: even rows get warm-50 bg
  • ☐ Bordered: all cells have visible borders
  • ☐ Minimal: no container, bottom borders only
  • ☐ Compact: 8px 12px padding
  • ☐ Comfortable: 16px padding
