2. Responsive Behavior

Strategies for handling tables on smaller screens while maintaining data accessibility.

Horizontal Scroll

Wrap table in scrollable container. Use overflow-x: auto with smooth scrolling enabled.

Column A Column B Column C
Data 1 Data 2 Data 3

Priority Columns

Hide low-priority columns on small screens. Use CSS classes to control visibility at breakpoints.

Priority 1: Always visible (ID, Name, Status)
Priority 2: Hide below md (Date, Amount)
Priority 3: Hide below lg (Description)

Card View Alternative

Transform rows into stacked cards on mobile. Each card shows all data in a label-value format.

ID #TXN-001
Amount $125.00
Status Complete

Sticky First Column

Keep identifier column visible while scrolling horizontally. Uses position: sticky with left: 0.

ID Column B Column C
#001 Data 2 Data 3

4. Code Reference

CSS implementation for pagination and responsive table patterns.

Pagination Styles

/* Pagination container */ .pagination { display: flex; align-items: center; gap: 8px; } /* Pagination button */ .pagination-btn { padding: 6px 12px; border: 1px solid hsl(30 5% 88%); border-radius: 6px; font-size: 14px; cursor: pointer; transition: all 0.15s ease; } /* Active state - indigo */ .pagination-btn.active { background: hsl(230 28% 42%); border-color: hsl(230 28% 42%); color: white; }

Responsive Patterns

/* Horizontal scroll container */ .table-responsive { overflow-x: auto; -webkit-overflow-scrolling: touch; } /* Sticky first column */ .sticky-first-col td:first-child, .sticky-first-col th:first-child { position: sticky; left: 0; z-index: 10; background: inherit; } /* Card view for mobile */ @media (max-width: 640px) { .table-card { display: block; } .table-card-hide { display: none; } }

5. Full Implementation Checklist

Complete specifications to verify when implementing tables.

Container

  • ☐ Border radius: 8px
  • ☐ Border: 1px warm-200
  • ☐ Background: white (light)
  • ☐ Background: intaglio-800 (dark)
  • ☐ Overflow: hidden

Header Styling

  • ☐ Font size: 12px
  • ☐ Font weight: 600
  • ☐ Text transform: uppercase
  • ☐ Letter spacing: 0.05em
  • ☐ Padding: 12px
  • ☐ Background: warm-50 (light)

Row States

  • ☐ Hover: warm-50 background
  • ☐ Selected: indigo-50 bg + left border
  • ☐ Border left: 3px indigo-500
  • ☐ Disabled: 50% opacity
  • ☐ Cell padding: 12px

Interactive Features

  • ☐ Sortable: cursor pointer
  • ☐ Sort icon: 40% opacity default
  • ☐ Sort icon hover: 100% opacity
  • ☐ Checkbox: 18px, 4px radius
  • ☐ Checked: indigo-500 fill

Responsive Behavior

  • ☐ Horizontal scroll enabled
  • ☐ Touch scrolling smooth
  • ☐ Custom scrollbar styled
  • ☐ Priority columns defined
  • ☐ Card view at 640px

Accessibility

  • ☐ role="grid" on table
  • ☐ aria-sort on headers
  • ☐ aria-selected on rows
  • ☐ Arrow key navigation
  • ☐ Focus visible indicators
  • ☐ Screen reader announcements
