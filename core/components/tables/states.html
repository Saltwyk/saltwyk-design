6. Code Reference

CSS implementation for row states, status badges, and loading states.

Row States

/* Hover state */ .row-hover { background: hsl(30 5% 98%); } .row-hover-dark { background: hsl(24 28% 18%); } /* Selected state (indigo) */ .row-selected { background: hsl(230 25% 96%); border-left: 3px solid hsl(230 28% 42%); } .row-selected-dark { background: hsl(230 28% 42% / 10%); border-left: 3px solid hsl(230 26% 54%); } /* Active/Pressed state */ .row-active { background: hsl(30 5% 95%); } .row-active-dark { background: hsl(24 29% 12%); } /* Disabled state */ .row-disabled { opacity: 0.5; } .row-disabled td { color: hsl(30 6% 62%); }

Status Badges

/* Badge base */ .status-badge { display: inline-flex; align-items: center; gap: 6px; padding: 4px 10px; border-radius: 9999px; font-size: 12px; font-weight: 500; } .status-badge .dot { width: 6px; height: 6px; border-radius: 50%; } /* Light mode variants */ .status-success { background: hsl(150 45% 92%); color: hsl(150 100% 22%); } .status-warning { background: hsl(20 50% 93%); color: hsl(20 80% 30%); } .status-error { background: hsl(330 50% 94%); color: hsl(330 85% 35%); } .status-pending { background: hsl(74 45% 90%); color: hsl(74 80% 28%); } .status-info { background: hsl(237 45% 92%); color: hsl(237 100% 40%); }

Skeleton Loading

/* Skeleton with shimmer animation */ .skeleton { background: linear-gradient(90deg, hsl(30 5% 93%) 25%, hsl(30 5% 97%) 50%, hsl(30 5% 93%) 75%); background-size: 200% 100%; animation: skeleton-shimmer 1.5s infinite; border-radius: 4px; height: 16px; } @keyframes skeleton-shimmer { 0% { background-position: 200% 0; } 100% { background-position: -200% 0; } }

7. Implementation Checklist

Key specifications to verify for table states.

Row States

  • ☐ Hover: warm-50 (light) / intaglio-700 (dark)
  • ☐ Selected: indigo-50 + 3px left border
  • ☐ Active: warm-100 (light) / intaglio-800 (dark)
  • ☐ Disabled: 50% opacity + warm-400 text

Column Alignment

  • ☐ Text columns: left aligned
  • ☐ Numbers: right aligned + tabular-nums
  • ☐ Dates: right aligned
  • ☐ Status: centered
  • ☐ Actions: right aligned

Status Badges

  • ☐ Dot indicator: 6px circle
  • ☐ Padding: 4px 10px
  • ☐ Font: 12px / 500 weight
  • ☐ Border radius: 9999px (pill)

Empty & Loading States

  • ☐ Empty: 48px padding, 48px icon
  • ☐ Skeleton height: 16px
  • ☐ Shimmer animation: 1.5s infinite
  • ☐ Skeleton widths: vary 50%-85%
