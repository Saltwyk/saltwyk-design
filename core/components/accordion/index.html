Component

Accordion

Collapsible content panels for organizing information into expandable sections. Used for email groups in Settings and other disclosure patterns.

Overview

Interaction

Click header to toggle expand/collapse

Animation

Chevron rotates 0° → 90° on expand

Accessibility

ARIA expanded, controls, keyboard nav

Default State

Collapsed by default

Accessibility

ARIA Attributes

  • aria-expanded - true/false on trigger button
  • aria-controls - ID of content panel
  • id - Unique ID on content panel

Keyboard Support

  • Enter / Space - Toggle expand/collapse
  • Tab - Move focus between triggers
  • Focus ring: emerald double-ring pattern

Usage Guidelines

Do

  • • Use bordered variant for grouped list items
  • • Use warning variant for items needing action
  • • Keep accordion headers concise and scannable
  • • Allow multiple items to be open simultaneously

Don't

  • • Nest accordions more than one level deep
  • • Use for content that should always be visible
  • • Hide critical information in collapsed state
  • • Use basic variant for card-style layouts

Code Example

<div class="accordion-item accordion-bordered"> <button class="accordion-trigger" aria-expanded="false" aria-controls="content-1"> <i data-lucide="chevron-right" class="accordion-chevron"></i> <span class="font-medium text-ink flex-1">Header Text</span> <span class="badge-verified">Verified</span> </button> <div id="content-1" class="accordion-content" hidden> <div class="accordion-content-inner"> <!-- Content here --> </div> </div> </div>
