Accessibility

ARIA attributes and keyboard navigation requirements.

ARIA Attributes

  • • Tab list: role="tablist"
  • • Tab trigger: role="tab", aria-selected, aria-controls
  • • Tab panel: role="tabpanel", aria-labelledby
  • • Inactive tabs: tabindex="-1"
  • • Active tab: tabindex="0"

Keyboard Navigation

  • • ← / → — Navigate between tabs (horizontal)
  • • ↑ / ↓ — Navigate between tabs (vertical)
  • • Home — First tab
  • • End — Last tab
  • • Tab — Move to panel content

Focus Management

  • • Focus should move to the active tab when entering the tab list
  • • Arrow keys move focus between tabs without activating them (optional: automatic activation)
  • • Enter/Space activates the focused tab
  • • Focus ring: 2px emerald-500 outline with 2px offset
