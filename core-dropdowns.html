Accessibility

ARIA attributes and keyboard navigation.

ARIA Attributes

  • • Trigger: role="combobox", aria-expanded, aria-haspopup="listbox"
  • • Menu: role="listbox"
  • • Option: role="option", aria-selected
  • • Disabled option: aria-disabled="true"

Keyboard Navigation

  • • Enter / Space — Open menu, select option
  • • Escape — Close menu
  • • ↑ / ↓ — Navigate options
  • • Home / End — First / last option
  • • Type-ahead — Jump to matching option
