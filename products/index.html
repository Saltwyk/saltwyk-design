Product Design

Product Surfaces

Each Saltwyk product surface applies the Opal Framework differently. The same foundation tokens express distinct personalities through three opal treatments.

Opal Treatments

The Opal Framework defines how thread colors and surfaces combine. Each treatment serves a different user context:

Bright Opal

Light backgrounds, soft thread tints. Approachable and consumer-friendly.

Fire Opal

Dark intaglio backgrounds, bright thread accents. Professional authority.

Bright Opal

Minimal, clean surfaces. Emerald only for interactive states.
