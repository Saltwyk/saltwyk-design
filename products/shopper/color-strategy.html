Full Saturation = Semantic Meaning

Thread colors at full saturation (400-600 range) are reserved for semantic states. Never use them for purely decorative purposes—users will interpret them as status indicators.

Cyan (full)

= Info

Emerald (full)

= Success / Interactive

Orange (full)

= Warning

Magenta (full)

= Error / Destructive

Never use full-saturation thread colors (400-600 range) for decorative purposes. Users will interpret them as status indicators. A decorative card with a cyan-500 thread looks like an "info" state.

Validation

Review Checklist

  • Decorative threads use 100-300 range (tints), never full saturation
  • Full saturation (400-600) only for semantic meaning (Info, Success, Warning, Error)
  • Dark mode threads use 300-500 range (shades)
  • Emerald only appears on interactive elements
  • Maximum one hero thread per page
  • Thread accents used on 1-2 featured elements, not all
  • Users cannot confuse decorative threads with status indicators
  • Page feels warm and approachable, not corporate
