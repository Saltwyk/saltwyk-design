Screen Specification

Profile Dropdown

Context-aware dropdown menu for profile avatar. Shows current role, growth CTAs, and context switching for dual-role users.

Preview Mode

Overview

The Profile Dropdown appears when clicking the profile avatar in the top navigation. It adapts based on the user's profile configuration:

  • Shopper Only: Shows shopping context + CTA to add professional profile
  • Professional Only: Shows work context + CTA to add shopping profile
  • Both Profiles: Shows current context + switch option (no growth CTA)

Mobile Layout - All Variants

Dropdown width is 280px on mobile with 8px padding. Positioned below the profile avatar.

Behavior Notes

Growth CTA Removal

The "Use Saltwyk for Business" / "Shop with Points" CTA only appears when the user has a single profile type. Once they activate both profiles, the growth CTA is replaced by the context switch UI.

Context Switch Behavior

Clicking the switch card triggers an instant context change. No modal, no loading spinner. The sidebar theme transitions (light ↔ dark), URL updates, and content loads seamlessly.

Contact Info Priority

Shopping context: Phone number is primary (OTP login).
Professional context: Work email is primary, phone secondary.

Avatar Color

Shopping context: Warm gray avatar (bg-warm-300).
Professional context: Emerald avatar (bg-emerald-600) indicating active work mode.
