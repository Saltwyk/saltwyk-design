Shopper Component

Wishlist Item

Display wishlist item with merchant logo, product title, and progress toward affordability. Shows Ready to Snag or Saving For states based on point balance.

Progress Ring Calculation

// For a 24px ring (viewBox 0 0 24 24) // radius = 10, circumference = 2 * PI * 10 = 62.83 const circumference = 62.83; const progress = 0.75; // 75% const dashoffset = circumference * (1 - progress); // dashoffset = 62.83 * 0.25 = 15.71 // SVG attributes: // stroke-dasharray="62.83" // stroke-dashoffset="15.71"

Logo.dev Integration

Merchant Logos

Use logo.dev API for merchant logos with fallback to colored initials. Logo variant uses 64px, photo variant uses 24px:

<!-- Logo Variant: 64px in product position --> <div class="w-16 h-16 flex-shrink-0"> <img src="https://img.logo.dev/sephora.com?token=pk_YOUR_TOKEN&size=64&format=png" alt="Sephora" class="w-16 h-16 rounded-full object-cover shadow-sm" onerror="this.style.display='none'; this.nextElementSibling.style.display='flex'" /> <div class="w-16 h-16 rounded-full bg-sepia-400 text-white flex items-center justify-center text-xl font-bold shadow-sm" style="display:none"> SE </div> </div>

Note: Token should be loaded from environment variable in production. Placeholder pk_YOUR_TOKEN used in this spec.

Props API

React Implementation

TypeScript Interface

interface WishlistItemProps { status: 'ready' | 'saving'; title: string; merchantName: string; merchantDomain: string; // e.g., "sephora.com" for logo.dev merchantInitials: string; // Fallback initials productImageUrl?: string; // If provided, use photo variant progress: number; // 0-100 for saving state onSnag?: () => void; // only for ready state } // productImageUrl: // - When provided: Renders photo variant (full-bleed image at top) // - When null/undefined: Renders logo variant (64px logo in product position)

Color Semantics

Critical Design Rules

Ready to Snag State

  • • NO thread (secondary card)
  • • Emerald check-circle icon + status text
  • • Outlined indigo CTA (secondary style)
  • • 64px logo or full-bleed photo

Saving For State

  • • NO thread accent (not ready yet)
  • • Indigo progress ring
  • • NO CTA button
  • • NO price or points (simplified)
