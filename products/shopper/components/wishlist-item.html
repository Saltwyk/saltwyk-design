Progress Ring Calculation

// For a 36px ring (viewBox 0 0 36 36) // radius = 16, circumference = 2 * PI * 16 = 100.53 const circumference = 100.53; const progress = 0.75; // 75% const dashoffset = circumference * (1 - progress); // dashoffset = 100.53 * 0.25 = 25.13 // SVG attributes: // stroke-width="3" // stroke-dasharray="100.53" // stroke-dashoffset="25.13" // class="-rotate-90" (Tailwind) // Track (background) circle: stroke="hsl(var(--indigo-200))" // Progress arc: stroke="hsl(var(--indigo-500))"

Critical Design Rules

Quality Standards

Do

  • • Use 2px borders on all cards
  • • Add hover states (border-indigo-300 + shadow-lg)
  • • Emerald CTA for Ready state ("Snag It Now!")
  • • Outlined indigo CTA for Saving state
  • • Logo gets ring-2 ring-white for separation
  • • White header with 2px border (not colored)

Don't

  • • NO emerald headers (clashes with CTA)
  • • NO 1px borders (feels cheap)
  • • NO shadow-sm (use shadow-md minimum)
  • • NO filled CTAs on Saving state (use outlined)
  • • NO py-2 on banners (use py-3)
  • • NO skipping "Ways to Earn More" CTA

Why emerald CTAs on every card?

"Snag It Now!" is the single most important action across the entire Shopper experience. Using emerald reinforces that every ready item is a genuine opportunity—multiple CTAs don't compete, they celebrate availability.

Earning Pathways Modal

"Ways to Earn More" opens a modal showing cashback opportunities, bonus offers, and activities to accelerate progress toward the item. This drives engagement and helps users reach Ready state faster.
