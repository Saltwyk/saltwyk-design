Logo.dev Integration

Merchant Logos

Use logo.dev API for merchant logos with automatic fallback to colored initials:

<div class="relative"> <img src="https://img.logo.dev/sephora.com?token=pk_YOUR_TOKEN&size=48&format=png" alt="Sephora" class="w-12 h-12 rounded-full object-cover border-2 border-ledger" onerror="this.style.display='none'; this.nextElementSibling.style.display='flex'" /> <!-- Fallback --> <div class="w-12 h-12 rounded-full bg-sienna-400 text-white flex items-center justify-center text-sm font-semibold border-2 border-ledger" style="display:none"> SE </div> </div>

Logo.dev API: Token should be loaded from environment variable in production. Placeholder pk_YOUR_TOKEN used in this spec.

Color Hash Algorithm

function getMerchantColor(name: string): string { const hash = name.split('').reduce((acc, char) => acc + char.charCodeAt(0), 0); const colors = ['bg-sepia-400', 'bg-sienna-400', 'bg-slate-400', 'bg-warm-400']; return colors[hash % colors.length]; } function getInitials(name: string): string { // Handle numeric prefixes: "1-800-Flowers" → "18" const numericMatch = name.match(/^\d+/); if (numericMatch) return numericMatch[0].slice(0, 2); // Standard: first letter of first two words const words = name.split(/[\s-]+/); if (words.length >= 2) return (words[0][0] + words[1][0]).toUpperCase(); // Single word: first two characters return name.slice(0, 2).toUpperCase(); }

Usage Guidelines

When to Use

  • • Display connected merchant loyalty accounts
  • • Show merchant count at a glance in cards/summaries
  • • Compact representation of multiple merchants
  • • Use with "View all" link for full list access

Accessibility

  • • Add alt attribute with merchant name to each logo
  • • Make overflow indicator a <button> element
  • • Include aria-label="View all X merchants" on overflow
  • • Ensure fallback initials have sufficient color contrast
