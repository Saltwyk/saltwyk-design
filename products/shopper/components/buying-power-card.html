Shopper Component

Buying Power Card

Hero card displaying the shopper's buying power status with appropriate visual treatment and call-to-action. Shows Low, Building, or Ready states based on connected merchants.

Component States

All Three States

State: Low (Default)

LOW BUYING POWER

Let's build it up

Connect your favorite merchants to start earning.

Visual Treatment

  • thread: ON TOP EDGE (square corners above)
  • card body: bg-ledger rounded-b-2xl
  • headline: Gradient text (intaglio)
  • status dot: bg-magenta-500 + pulse

Actions

  • primary CTA: Emerald button
  • This is the ONLY emerald usage

State: Building

BUILDING BUYING POWER

You're on your way

3 merchants connected. Connect more to increase power.

View Merchants →

Visual Treatment

  • thread: ON TOP EDGE (square corners above)
  • card body: Fresh gradient @ 20% rounded-b-2xl
  • headline: Gradient text (intaglio)
  • status dot: bg-indigo-500 (no pulse)

Actions

  • secondary action: Indigo link (NOT emerald)
  • No primary CTA in this state

State: Ready

READY TO SHOP

Your buying power is strong

7 merchants connected • Active

Browse Wishlist →

Visual Treatment

  • thread: ON TOP EDGE (square corners above)
  • card body: Fresh gradient @ 40% rounded-b-2xl
  • headline: Gradient text (intaglio)
  • status dot: bg-emerald-500 (no pulse)

Actions

  • secondary action: Indigo link
  • Status dot is emerald (decorative, not CTA)

Color Semantics

Critical Design Rules

Correct Usage

  • • Emerald for ONE primary CTA (Low state only)
  • • Emerald status dot (Ready state - decorative)
  • • Indigo for all interactive links
  • • Cyan ONLY in thread gradient (never standalone)
  • • Fresh thread at top of ALL states

Never Do This

  • • Multiple emerald buttons in same card
  • • Emerald links instead of buttons
  • • Cyan as standalone color (threads only)
  • • Primary CTA in Building/Ready states
  • • Missing thread accent at top

Responsive
