5. Code Reference

CSS class definitions for top navigation components.

Container Styles

.topnav {
  height: 64px;
  background: white;
  border-bottom: 1px solid hsl(30 5% 88%);
  display: flex;
  align-items: center;
  padding: 0 24px;
}

.topnav-dark {
  background: hsl(24 29% 8%);
  border-bottom-color: hsl(24 28% 18%);
}

Nav Item Styles

.topnav-item {
  padding: 8px 16px;
  font-size: 14px;
  font-weight: 500;
  color: hsl(30 6% 46%);
  border-bottom: 2px solid transparent;
  margin-bottom: -1px;
  height: 64px;
  display: flex;
  align-items: center;
  transition: all 0.15s ease;
}

.topnav-item:hover {
  color: hsl(24 29% 8%);
}

.topnav-item-active {
  color: hsl(24 29% 8%);
  border-bottom-color: hsl(237 100% 65%); /* indigo-500 */
}

Dark Mode Nav Items

.topnav-item-dark {
  color: hsl(30 5% 55%);
}

.topnav-item-dark:hover {
  color: hsl(30 5% 85%);
}

.topnav-item-dark.topnav-item-active {
  color: hsl(30 5% 95%);
  border-bottom-color: hsl(237 90% 72%); /* indigo-400 */
}

6. Implementation Checklist

Container

  • ☐ Height: 64px
  • ☐ Light: white background, warm-200 border
  • ☐ Dark: intaglio-900 background, intaglio-700 border
  • ☐ Horizontal padding: 24px

Nav Items

  • ☐ Font: 14px, 500 weight
  • ☐ Padding: 8px 16px
  • ☐ Full height (64px) with centered content
  • ☐ 150ms transition on color change

Active State

  • ☐ Light: indigo-500 underline (2px)
  • ☐ Dark: indigo-400 underline (2px)
  • ☐ Underline overlaps bottom border
  • ☐ Text color matches hover state

Accessibility

  • ☐ Use <nav> semantic element
  • ☐ aria-current="page" on active item
  • ☐ Keyboard navigation with Tab key
  • ☐ Focus visible ring (indigo-500)
