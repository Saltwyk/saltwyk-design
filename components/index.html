Components

Reusable UI building blocks for all product surfaces

Components are the building blocks of Saltwyk interfaces. Built on foundation tokens, they provide consistent interaction patterns across all product surfaces.

Each component is designed to work in both Bright Opal (light) and Fire Opal (dark) treatments.
