5. Code Reference

CSS implementation for multi-select components.

Selection Tags

.select-tag { @apply inline-flex items-center gap-1; @apply px-2 py-0.5 rounded; @apply bg-indigo-100 text-indigo-700; @apply text-xs font-medium; } .select-tag-remove { @apply w-3.5 h-3.5 cursor-pointer; @apply text-indigo-500 hover:text-indigo-700; }

Checkbox

.dropdown-checkbox { @apply w-4 h-4 rounded; @apply border-[1.5px] border-warm-400; @apply flex items-center justify-center; @apply transition-colors duration-100; } .dropdown-checkbox-checked { @apply bg-indigo-600 border-indigo-600; } .dropdown-checkbox-checked svg { @apply w-3 h-3 text-white; }

6. Implementation Checklist

Key specifications to verify for multi-select components.

Trigger

  • ☐ Min height: 44px (with tags)
  • ☐ Tags wrap to new line
  • ☐ Max visible tags (then "+N")
  • ☐ Remove tag on X click

Menu

  • ☐ Checkbox: 16x16px
  • ☐ Select All / Clear actions
  • ☐ Stay open on selection
  • ☐ Close on click outside
