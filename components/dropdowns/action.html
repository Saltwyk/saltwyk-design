4. Code Reference

CSS implementation for menu containers and items.

Menu Container

/* Light mode */ .dropdown-menu { @apply absolute z-50; @apply bg-white border border-warm-200 rounded-lg; @apply py-1 min-w-[180px] max-w-[320px]; @apply overflow-auto max-h-[320px]; box-shadow: 0 4px 16px rgba(0,0,0,0.12); } /* Dark mode */ .dropdown-menu-dark { @apply bg-intaglio-800 border-intaglio-700; box-shadow: 0 4px 16px rgba(0,0,0,0.3); }

Menu Item

/* Light mode */ .dropdown-item { @apply flex items-center gap-2 px-3 py-2; @apply text-sm text-ink cursor-pointer; @apply transition-colors duration-100; } .dropdown-item:hover { @apply bg-warm-50; } .dropdown-item-destructive { @apply text-magenta-600; } .dropdown-item-destructive:hover { @apply bg-magenta-50; }

5. Implementation Checklist

Key specifications to verify for action menus.

Trigger Button

  • ☐ Size: 32x32px
  • ☐ Border radius: 6px
  • ☐ Hover: warm-50 background
  • ☐ Icon: 20x20px ellipsis

Menu Behavior

  • ☐ ARIA: role="menu"
  • ☐ Items: role="menuitem"
  • ☐ Destructive items separated by divider
  • ☐ Position: below trigger, flip if needed
