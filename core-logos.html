Usage Guidelines

Minimum Size

Lockup: 120px

Mark: 24px

Below these sizes, details become illegible.

Clear Space

Maintain clear space equal to the height of the logomark "S" around all sides.

Background Rules

✓ Solid backgrounds
✓ Subtle gradients
✗ Busy images
✗ Low contrast

Ensure sufficient contrast for legibility.

Incorrect Usage

Avoid these common misuses to maintain brand consistency.

✗

Don't rotate

✗

Don't stretch

✗

Don't reduce opacity

✗

Don't use on clashing colors
