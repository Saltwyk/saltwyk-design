Core Foundation

Thread Line Variants

The security thread motif is central to Saltwyk's visual identity.

Full Spectrum

Width Variants

Complete thread color cycle. Use for major section dividers and hero accents.

Heavy (3px) - thread-line-full

Primary usage. Hero sections, major dividers.

Medium (2px) - thread-line-medium

Section dividers, card accents.

Subtle (1px) - thread-line-subtle

Inline dividers, subtle accents.

Full Spectrum

Fade Variants

Thread lines that fade to transparent. Use for asymmetric layouts.

Fade Left - thread-line-fade-left

Starts transparent, reveals thread.

Fade Right - thread-line-fade-right

Thread fades to transparent.

Fade Both - thread-line-fade-both

Floating thread. Good for centered elements.
