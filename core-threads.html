Guidelines

Thread Placement Principles

Core rules for using threads effectively in layouts.

Do

  • One thread per page — A single thread creates a clear visual anchor
  • Place at the very top — Above the hero, full-bleed, as the first element
  • Use full width — Threads should span edge-to-edge, not be contained
  • Square top corners — Use rounded-b-lg on hero, not rounded-lg

Don't

  • Multiple threads per page — Competing threads dilute the visual anchor
  • Thread below hero — Creates confusion about what the thread is anchoring
  • Inset or contained threads — Threads are not decoration, they're structure
  • Round all corners — The thread needs to flow seamlessly into the hero

Exception: Vertical threads can be used within content for pull quotes or sidebar accents, as they serve a different structural purpose than the horizontal page anchor.
