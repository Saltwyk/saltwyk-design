Stress Test v3

Thread Assignment Stress Test

Testing threads in realistic hero contexts with background gradients. Introducing new BRIGHT thread options that are vibrant but NOT hot/warning colors.

Key Insight: Muted/cooler gradients don't work — threads need CONTRAST. But hot colors (orange, magenta) read as warning/error. We need bright threads using cyan, lime, indigo, and emerald — vibrant but not alarming.
