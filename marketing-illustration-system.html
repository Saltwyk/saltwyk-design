Marketing Design System

Illustration Prompting System

A systematic approach to generating consistent sepia intaglio illustrations with AI.

Philosophy

Illustrations reinforce Saltwyk's "currency aesthetic"—the look of money, security, and trust. We use a sepia intaglio engraving style reminiscent of banknote art: fine crosshatching, stippling, ornate borders, and allegorical subjects. This visual language says "established," "valuable," and "permanent" without words.

Unlike contemporary flat illustrations or photorealistic renders, our style evokes the craftsmanship of 19th-century engravers—the same tradition that still appears on the back of a $100 bill. This deliberate anachronism creates instant gravitas and distinguishes Saltwyk from fintech competitors chasing minimalist trends.

Illustrations are generated via AI (Google Gemini) using a systematic prompting approach. This document provides the vocabulary, structure, and quality criteria needed to produce consistent, on-brand imagery every time.

✓ When to Use Illustrations

  • • Hero sections (marketing website)
  • • Empty states (product UI)
  • • Pitch decks and investor materials
  • • Documentation headers (sparingly)
  • • Email headers and social cards

✗ When NOT to Use

  • • Product UI (too heavy, distracts from function)
  • • Checkout flow (competes for attention)
  • • Data-dense screens (competes with content)
  • • Small icons or UI elements

The Saltwyk Illustration Aesthetic

Visual Characteristics

Style Intaglio engraving, fine crosshatching, stippling
Color palette
Sepia monochrome
sepia-100 → sepia-700
Line weight Fine, precise, evenly spaced
Subjects Allegorical (scales, handshakes, commerce symbols, flowing forms)
Borders Ornate guilloche patterns, scrollwork frames, rosettes
Mood Trustworthy, established, valuable, permanent

What It Should Feel Like

  • ✓ The back of a $100 bill
  • ✓ A vintage stock certificate
  • ✓ A trust seal from a 19th-century bank
  • ✓ An antique bond certificate

What It Should NOT Feel Like

  • ✗ Cartoon or whimsical
  • ✗ Flat vector illustration
  • ✗ Modern minimalist line art
  • ✗ Photorealistic or 3D renders

Prompt Architecture

Base Prompt Structure

[STYLE] + [SUBJECT] + [COMPOSITION] + [DETAILS] + [EXCLUSIONS]

Each prompt combines these elements in sequence. STYLE and EXCLUSIONS remain constant; the middle elements vary by use case.

CONSTANT

In the style of vintage currency engraving, sepia-toned intaglio illustration with fine crosshatching and stippling, reminiscent of 19th-century banknote art. Monochromatic warm brown tones.

DETAILS

Additional specifications:

  • • Density: sparse / medium / dense

Quality Gates

Use these criteria to evaluate whether a generated illustration is on-brand.

✓ Accept If

  • ● Monochromatic sepia/brown tones only
  • ● Fine line work visible (not blurry or soft)
  • ● Crosshatching or stippling technique apparent
  • ● Subject is allegorical/symbolic, not literal
  • ● Feels like it could be on currency
  • ● No text or typography in image

✗ Reject If

  • ● Any color beyond sepia range
  • ● Soft, painterly, or photorealistic rendering
  • ● Cartoon or whimsical style
  • ● Modern objects (computers, phones, etc.)
  • ● Text or numbers visible
  • ● Flat vector appearance
  • ● Subject is too literal (photo of actual money)

Iteration Guidance

When outputs miss the mark, use these adjustments to steer generation.

If output is too modern:

Add:

"19th-century aesthetic, historical engraving style, antique"

Remove:

Any contemporary descriptors

If output is too colorful:

Add:

"Strictly monochromatic, sepia only, no color variation"

Emphasize:

"Warm brown tones ranging from cream to dark brown"

If output is too soft/painterly:

Add:

"Sharp precise lines, visible crosshatching, engraved look"

Add:

"High contrast between light and dark areas"

If output lacks detail:

Add:

"Intricate fine details, dense linework, elaborate ornamentation"

Specify:

"Density: Dense"

If composition is wrong:

Be more specific about framing:

"Centered in oval vignette" "Rectangular frame with 2-inch border"

Implementation Notes

File Formats

  • • Generate at high resolution (2048px minimum)
  • • Export as PNG with transparency where appropriate
  • • Create 1x and 2x versions for retina

Color Adjustment

  • • Apply color overlay using sepia-500 as midtone
  • • Ensure darkest areas don't exceed sepia-800
  • • Ensure lightest areas don't go below sepia-50

Placement Guidance

  • • Hero: Full width or 50% with text wrap
  • • Empty states: Centered, max 200px width
  • • Documentation: Corner ornaments, max 100px
  • • Pitch decks: Full bleed or contained in frame
