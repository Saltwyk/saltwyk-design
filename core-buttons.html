Buttons

Interactive elements for actions and navigation. Used across Product and Marketing contexts.

Common Variants

Universal button variants used across all Saltwyk surfaces.

Variant Background Text Border Usage
Primary bg-emerald-500 text-white none Main CTAs, form submissions
Secondary bg-warm-200 text-ink border-warm-300 Secondary actions, cancel
Ghost transparent text-ink none Subtle actions, toolbars
Outline transparent text-ink border-warm-300 Tertiary actions
Outline Primary transparent text-emerald-600 border-emerald-500 Alternative primary, less emphasis
Destructive bg-magenta-500 text-white none Delete, remove, danger actions

Product-Specific Variants

Additional variants for Shopper Passport and Merchant Dashboard contexts.

Variant Background Text Border Usage
Solid bg-intaglio-900 text-ledger none Primary product actions (dark theme)
Dashed transparent text-intaglio/60 border-dashed border-intaglio/30 Add items, optional actions

Choosing Between Primary and Solid

  • Primary (emerald): Use for marketing pages, checkout widget, and cross-platform contexts
  • Solid (intaglio): Use within Shopper Passport and Merchant Dashboard for a cohesive product feel

All Variants with Icons

Icon Guidelines

  • • Use 16px (w-4 h-4) icons for sm and md buttons
  • • Use 20px (w-5 h-5) icons for icon-only md buttons
  • • Use 24px (w-6 h-6) icons for lg buttons
  • • Gap between icon and text: gap-2 (8px)
  • • Icon-only buttons should have aria-label or title for accessibility

Full Width

Full-width buttons for forms and mobile contexts.

Use w-full class for full-width buttons. Common in mobile views and form submissions.

On Dark Backgrounds

Button treatments for dark hero sections and modals.

On dark backgrounds:

  • • Use white/inverted primary for maximum contrast
  • • Outline buttons use border-white/30
  • • Ghost buttons hover to bg-white/10
  • • Emerald primary still works when CTAs need brand color

Base Styles

.btn {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  gap: 0.5rem;           /* 8px */
  font-weight: 500;      /* medium */
  border-radius: 0.375rem; /* rounded-md */
  border: 1.5px solid transparent;
  transition: all 150ms ease;
  cursor: pointer;
}

.btn:focus-visible {
  outline: 2px solid var(--emerald-500);
  outline-offset: 2px;
}

.btn:disabled {
  opacity: 0.5;
  cursor: not-allowed;
  pointer-events: none;
}
