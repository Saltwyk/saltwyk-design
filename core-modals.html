Modals & Dialogs

Modal patterns for confirmations, forms, alerts, and focused interactions.

Modal Anatomy

Modal Title

Modal body content goes here. This area contains the main content of the modal, whether it's a form, information, or other interactive elements.

Structure

  • Backdrop: intaglio-900/80 (80% opacity)
  • Container: bg-white, rounded-lg, shadow-xl
  • Header: Title in H3, optional close button
  • Body: Content with p-6 padding
  • Footer: Actions, right-aligned

Spacing

  • Header: py-5 px-6
  • Body: p-6
  • Footer: py-4 px-6
  • Button gap: gap-3

Fire Opal Modal Tokens

  • • Container: bg-intaglio-800
  • • Border: 1px intaglio-700
  • • Text: warm-100 titles, warm-400 body
  • • Backdrop: intaglio-950/90

Animation

Entrance

  • Backdrop: fadeIn 150ms ease-out
  • Container: fadeInUp 200ms ease-out
  • Backdrop animates first, modal follows

Exit

  • Both: fadeOut 150ms ease-in
  • Modal and backdrop fade together

Accessibility

Focus Management

  • • Trap focus within modal while open
  • • First focusable element receives focus on open
  • • Return focus to trigger element on close
  • • Tab cycles through modal elements only

Keyboard & ARIA

  • • ESC key closes modal
  • • aria-modal="true" on container
  • • role="dialog" on container
  • • aria-labelledby referencing title
