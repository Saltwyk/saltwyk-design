Design System → Foundation

Foundation

Shared tokens across all product surfaces

Foundation defines the raw materials: color scales, type families, spacing units, motion curves. These tokens are shared across every surface—Shopper, Merchant, Marketing, and Docs.

What differs between surfaces is composition—how tokens combine. Fire opal and bright opal treatments use the same thread colors at different saturation levels. The foundation provides the palette; surfaces determine the application.

Token Categories

Colors

Warm gray, intaglio brown, emerald green, and four thread accent scales. Named surfaces: ledger and ink.
Aa

Typography

DM Serif Display for headlines, Inter for body, Outfit for brand moments. Type scale from caption to display.
Button

Buttons

Primary, secondary, ghost, outline variants. Three sizes. States including loading and disabled.

Logos

Lockup, logomark, and wordmark variants. Emerald, inverse, and monochrome options with usage guidelines.

Thread Lines

Security thread patterns. Full spectrum, single color accents, horizontal and vertical orientations, fade variants.

Gradients

Four thread families (cool, fresh, flame, tropic) at five intensity levels from thread to background.

Spacing & Layout

Base-4 spacing scale, container widths, grid patterns, and responsive breakpoints.

Motion & Animation

Duration scale, easing curves, entrance animations, continuous animations, and accessibility considerations.
