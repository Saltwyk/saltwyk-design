Core Foundation

Typography

Font families, type scale, and usage guidelines for product interfaces.

Font Families

DM Serif Display

Editorial serif for headlines and display text. Brings warmth and sophistication.

font-display

Fallback: Georgia, serif

Inter

Primary UI font. Excellent legibility at all sizes. Use for body, labels, and navigation.

font-body or font-sans

Fallback: system-ui, sans-serif

Outfit

Brand font. Reserved for logo lockups and special brand moments only.

font-brand

Fallback: system-ui, sans-serif

Usage Guidelines

DM Serif Display

  • ✓ Page titles and hero headlines
  • ✓ Section headers (H1, H2)
  • ✓ Marketing display text
  • ✓ Pull quotes and testimonials (italic)
  • ✗ Never for body text or UI labels
  • ✗ Never for buttons or navigation

Inter

  • ✓ All body text and paragraphs
  • ✓ Buttons, labels, form fields
  • ✓ Navigation and menus
  • ✓ H3, H4 in product contexts
  • ✓ Tables, lists, metadata
  • ✓ Captions and helper text

Font Weights

Inter Weights

Light (300) font-light
Regular (400) font-normal
Medium (500) font-medium
Semibold (600) font-semibold
Bold (700) font-bold

Weight Usage

  • Regular (400) — Body text, descriptions
  • Medium (500) — Labels, subtle emphasis
  • Semibold (600) — Buttons, headings, strong emphasis
  • Bold (700) — Rarely used; very strong emphasis only
  • Light (300) — Large display text only, never for body

Line Height

Class Value Use Case
leading-none 1.0 Large display headlines only
leading-tight 1.25 Headings (H1-H3)
leading-snug 1.375 Subheadings, short paragraphs
leading-normal 1.5 Default body text
leading-relaxed 1.625 Long-form reading (articles, docs)
