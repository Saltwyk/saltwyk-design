Page Specification

Contact Page

Single-column contact form. Uses XL input variants (56px height, 18px font) on desktop. Inline validation with magenta-500 error states. Success state replaces form in-place with calendar booking CTA linking to calendar.html.

Contact Page Design Decisions

  • Single column layout: All fields stack vertically. Max-width 560px on desktop for comfortable reading width. Centered on page.
  • XL inputs on desktop: 56px height, 18px font size. Reverts to lg (48px/16px) on mobile for better touch targets without overwhelming small screens.
  • Validation: Inline under fields using magenta-500 border + error message. Real-time for email format, on-submit for required fields.
  • Success state: Replaces form with emerald-bordered confirmation card. Checkmark icon, "Message Sent" headline, 48-hour response copy, and a primary button linking to calendar.html for booking.
  • Calendar flow: Calendly lives on its own page (calendar.html). Post-booking redirects to meeting-confirmed.html. No calendar embed on this page.
  • "Let's Talk" nav CTA: Links to this page. On this page, the nav CTA scrolls to the form or is visually de-emphasized.
