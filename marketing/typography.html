Marketing Design System

Marketing Typography

Expressive type hierarchy for Galaxy Opal

Marketing typography pushes further than product UI - larger scales, dramatic weight contrast, and sophisticated use of Fraunces to create memorable brand moments. This system balances impact with readability.

Philosophy

Display-First Approach

Marketing inverts the product hierarchy. While product UI uses Inter primarily with Fraunces accents, marketing leads with Fraunces display typography and uses Inter as support. This creates distinctive brand voice and visual drama.

Product UI Hierarchy

  • Primary Inter (body, labels, utility)
  • Accent Fraunces (headlines only)
  • Scale Conservative (max 48px)
  • Weights 400-600
  • Purpose Clarity, efficiency

Marketing Hierarchy

  • Primary Fraunces (headlines, emphasis)
  • Support Inter (body, descriptions)
  • Scale Dramatic (32px-96px)
  • Weights 300-700
  • Purpose Impact, memorability

Variable Font

Variable Font Optical Sizing

Fraunces is a variable font with optical sizing axis (opsz 9-144). Correct optical size settings dramatically improve readability and visual quality at different scales.

Incorrect - No Optical Sizing

The trillion point economy

font-size: 48px;

Default opsz may not be optimal for this size

Correct - Matched Optical Sizing

The trillion point economy

font-size: 48px; font-variation-settings: 'opsz' 72;

Letterforms optimized for this viewing size

Optical Sizing Rules

Display Size Optical Size Setting Code
96pt, 72pt, 64pt 144 font-variation-settings: 'opsz' 144
56pt, 48pt, 40pt 72 font-variation-settings: 'opsz' 72
32pt and below 9 font-variation-settings: 'opsz' 9

Why Optical Sizing Matters

  • Large sizes (144): Thinner strokes, more refined details, elegant contrast
  • Medium sizes (72): Balanced proportions, versatile readability
  • Small sizes (9): Heavier strokes, better legibility, more robust letterforms

Always set optical size explicitly - don't rely on defaults.

Mixed Weight Rules

  • ✓ Use for short headlines (1-2 lines max)
  • ✓ Alternate light (300) + bold (700) for maximum contrast
  • ✓ Keep all text same size and leading
  • ✓ Emphasize 2-3 key concepts, de-emphasize connectors
  • ✓ Test readability - should feel natural, not gimmicky
  • ✗ Don't mix more than 2 weights in one headline
  • ✗ Don't use in long-form content
  • ✗ Don't combine with italic (too much variation)

Spacing

Spacing Specifications

Letter spacing (tracking) and line height (leading) are critical for readability and visual quality.

Tracking Guide by Size

96pt -0.025em to -0.03em Tight - creates drama
72pt-64pt -0.02em to -0.025em Tight - maintains impact
56pt-48pt -0.015em to -0.02em Moderate - balanced
40pt-32pt -0.01em to -0.015em Subtle - readability
Below 32pt 0em (default) No negative tracking

Leading Guide by Context

Display headlines 1.05 - 1.15 Tight for impact
Section headlines 1.15 - 1.25 Moderate
Body text 1.5 - 1.6 Generous for readability
Testimonials (italic) 1.5 - 1.7 Extra space for italic
Small text (<16px) 1.5 - 1.6 More space at small sizes

Mobile Scale Adjustments

Marketing typography must scale appropriately for mobile. Never just shrink - adjust proportions and hierarchy.

Mobile-Specific Adjustments

  • • Leading: Increase by 0.05-0.1 (more breathing room on small screens)
  • • Tracking: Reduce negative tracking slightly (letterforms more legible)
  • • Spacing: Increase space between elements (touch targets)
  • • Line length: Limit to 60-70 characters

Fix: Only use italic in weights 300-400

Insufficient Contrast
