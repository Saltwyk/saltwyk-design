Marketing Design System

Galaxy Opal

Marketing's vibrant, expressive visual language

Galaxy Opal is the marketing surface treatment—a more dramatic and attention-capturing evolution of Fire Opal. It combines traditional finance gravitas (intaglio, sepia) with modern currency innovation (security threads, vibrant gradients) to tell Saltwyk's story.

Philosophy

Traditional Finance + Modern Currency

Traditional Finance

  • • Intaglio printing
  • • Sepia tones
  • • Engraved aesthetics
  • • Parchment surfaces
  • • Formal composition

The Bridge

Galaxy Opal synthesizes both worlds—the gravitas of established finance with the energy of modern currency innovation.

Modern Currency

  • • Security threads
  • • Holographic elements
  • • Vibrant gradients
  • • Dynamic energy

Decorative Elements

Security Thread Usage

Two-Per-Page Baseline

Every marketing page starts with two thread positions as the baseline.

1

Header Position

Horizontal gradient line immediately below the navigation header. Spans full width using .thread-line utility (3px height).

2

Hero Card Top

Applied to the top edge of the hero card/section. Square top corners (no border-radius). Gradient flows horizontally across card top.

Thread Rules

  • • Two threads are the baseline for every marketing page
  • • Additional thread usage requires intentional design decision
  • • Thread families: Cool, Fresh, Tropic, Spectrum

Decision Framework

Choosing Thread Families

Spectrum

Full rainbow

Homepage, about page, brand moments

Use when you need the full brand expression. Maximum vibrancy and energy.

Cool

Cyan → Magenta

Technology, infrastructure, trust themes

Professional authority. Good for B2B messaging and technical credibility.

Fresh

Lime → Cyan

Growth, success, positive outcomes

Optimistic and forward-looking. Great for success stories and benefits.

Tropic

Cyan → Lime → Orange

Diversity, ecosystem, network effects

Breadth and variety. Good for partner pages and ecosystem messaging.

Action Color

Primary Action Color

Emerald

hsl(var(--emerald-500))

Use For:

  • • Primary CTAs: All main action buttons
  • • Links: Default link color
  • • Success States: Completion, verification, positive outcomes

✗ Never Use For:

  • • Decoration only
  • • Backgrounds without purpose
  • • Competing with thread gradients

Emerald should feel special and intentional. Overuse diminishes its impact as an action color.

Standards

Color Contrast & Legibility

Minimum Contrast Requirements

Text on ledger background Must meet WCAG AA (4.5:1 for body, 3:1 for large text)
Text on dark intaglio Must use hsl(var(--warm-100)) or lighter
Emerald buttons White text meets AAA contrast
Thread gradients Decorative only—never carry meaning alone

Text on Ledger

Body text uses warm-600 or darker for sufficient contrast.

4.5:1 ✓ WCAG AA Pass

Text on Intaglio

Body text uses warm-100 to warm-300 for legibility.

7:1 ✓ WCAG AAA Pass
