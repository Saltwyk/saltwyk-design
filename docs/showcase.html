API Reference

Endpoint documentation with method badges and parameters

POST /v1/payments

Create a new payment to a merchant on the Saltwyk network.

Request Body

amount required integer

Amount in the smallest currency unit (e.g., cents for USD).
currency required string

Three-letter ISO 4217 currency code.
merchant_id required string

The ID of the merchant receiving the payment.

Response

{
  "id": "pay_1234567890",
  "object": "payment",
  "amount": 2500,
  "currency": "usd",
  "status": "succeeded",
  "merchant_id": "merch_abc123",
  "created": 1704067200
}

Composition 02

Code Sample

Multi-language code example with tab navigation

Create a Payment

const saltwyk = require('saltwyk')('sk_live_...');

const payment = await saltwyk.payments.create({
  amount: 2500,
  currency: 'usd',
  merchant_id: 'merch_abc123',
  metadata: {
    order_id: 'order_12345'
  }
});

console.log(payment.id);

Tip: Use idempotency_key to safely retry requests without creating duplicate payments.
