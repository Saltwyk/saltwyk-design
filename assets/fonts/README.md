# Saltwyk Typography

## Fonts

- **Outfit**: Used for wordmark and display text
  - License: SIL Open Font License 1.1
  - Source: https://fonts.google.com/specimen/Outfit
  - Weights in use: Regular (400), Medium (500), Bold (700)

## Usage

- Wordmark: Outfit Medium or Regular
- Marketing headlines: Outfit Medium
- Product UI: Use system fonts (per design system)

## Installation

Web projects should load from Google Fonts:
\`\`\`html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link href="https://fonts.googleapis.com/css2?family=Outfit:wght@400;500;700&display=swap" rel="stylesheet">
\`\`\`
