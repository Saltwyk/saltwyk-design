2. Specifications

Detailed specifications for the bottom navigation component.

Property Value Notes
Position fixed bottom Always visible at bottom
Height 64px + env(safe-area-inset-bottom) Accounts for notch/home indicator
Background white Solid background for readability
Border warm-200 (top) 1px solid border
Shadow 0 -2px 8px rgba(0,0,0,0.06) Subtle upward shadow
Icon Size 24px Standard touch target icon
Label Font Size 10px Compact but readable
Icon-Label Gap 4px Tight vertical spacing
Default Color warm-400 Inactive state
Active Icon indigo-600 Selection color
Active Label indigo-700 Darker for contrast
Max Items 5 Recommended maximum

4. Code Reference

CSS styles for bottom navigation component.

/* Bottom Navigation Container */ .bottomnav { position: fixed; bottom: 0; left: 0; right: 0; height: 64px; padding-bottom: env(safe-area-inset-bottom, 0); background: white; border-top: 1px solid hsl(var(--warm-200)); box-shadow: 0 -2px 8px rgba(0,0,0,0.06); display: flex; align-items: center; justify-content: space-around; z-index: 100; } /* Navigation Item */ .bottomnav-item { display: flex; flex-direction: column; align-items: center; justify-content: center; gap: 4px; padding: 8px 12px; min-width: 64px; color: hsl(var(--warm-400)); cursor: pointer; transition: color 0.15s ease; } .bottomnav-item-icon { width: 24px; height: 24px; } .bottomnav-item-label { font-size: 10px; font-weight: 500; } /* Active State - Uses INDIGO (selection color) */ .bottomnav-item-active { color: hsl(var(--indigo-600)); } .bottomnav-item-active:hover { color: hsl(var(--indigo-700)); } .bottomnav-item-active .bottomnav-item-icon { color: hsl(var(--indigo-600)); } .bottomnav-item-active .bottomnav-item-label { color: hsl(var(--indigo-700)); font-weight: 600; }

5. Implementation Checklist

Key specifications to verify during implementation.

Container

  • ☐ Position: fixed bottom
  • ☐ Height: 64px + safe-area-inset
  • ☐ Background: white
  • ☐ Border: warm-200 top (1px)
  • ☐ Shadow: 0 -2px 8px rgba(0,0,0,0.06)
  • ☐ z-index: 100

Navigation Items

  • ☐ Icon size: 24px
  • ☐ Label size: 10px
  • ☐ Gap: 4px between icon and label
  • ☐ Min width: 64px per item
  • ☐ Max items: 5

States

  • ☐ Default: warm-400
  • ☐ Hover: warm-600
  • ☐ Active icon: indigo-600
  • ☐ Active label: indigo-700
  • ☐ Active label weight: 600

Accessibility

  • ☐ Use role="navigation"
  • ☐ Add aria-label to nav
  • ☐ Mark current with aria-current="page"
  • ☐ Ensure sufficient touch targets (48px min)
