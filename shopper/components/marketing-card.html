Marketing Card

High-impact hero promotional spots for dashboard "Drops" campaigns. Showstopper design, not utility.

Preview Mode

| Full Page

Overview

The MarketingCard is the hero promotional spot on the shopper passport dashboard. This is where "Drops" campaigns live — connection bonuses, shopping multipliers, referrals, partnerships, milestones. These cards should be visually stunning — the moments where Saltwyk's brand identity really shines.

Purpose

Hero promotional content, limited-time offers, special campaigns

Location

Passport dashboard hero section, above the fold

Thread Rule

This IS the hero — gets the spectrum thread accent

Visual Weight

Maximum impact — larger, bolder, full opalescence

Design Decision: Creative Flexibility, Not Rigid Rules

Marketing cards have creative flexibility rather than rigid rules. The surface + matching thread pattern is reliable and always works, but contrasting gradients, photo heroes, and experimental approaches can create surprise and emotional impact. Choose the pattern that best serves the campaign's goals.

Pattern Options:

  • • Surface + matching thread — reliable, cohesive
  • • Contrasting gradients — surprising, energetic
  • • Deep gradient + light text — bold, premium

 

  • • Photo hero + overlay — visceral, contextual
  • • Gradient text on solid — typography focus
  • • Experimental — push boundaries

CTA Button Rules

Button styling depends on background type. Always ensure high contrast.

On Gradient Backgrounds

bg-white text-emerald-700

On Light Backgrounds

bg-emerald-500 text-white

On Dark Backgrounds

bg-emerald-500 text-white

Never use black CTAs on marketing cards

Primary actions are always bg-emerald-500 or bg-white. The intaglio palette is for backgrounds, not buttons.

Usage Guidelines

Do

  • • Use full-intensity opalescent gradients
  • • Match thread color to campaign mood
  • • Use Fraunces 800-900 for headlines
  • • Keep no rounded top corners with thread
  • • Use white CTAs on gradient backgrounds
  • • Use emerald CTAs on light/dark backgrounds
  • • Include expiration/limitation info

Don't

  • • Don't use muted or surface-level gradients
  • • Don't use black or intaglio for CTAs
  • • Don't round top corners with thread present
  • • Don't use more than one marketing card per screen
  • • Don't use thin or light font weights
  • • Don't crowd the card with too much text
  • • Don't forget to show campaign end date
