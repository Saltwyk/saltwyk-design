Merchant Management Component

Empty State

Reusable component for empty and zero states across the shopper passport. Used when no items, merchants, or results are available.

Overview

The Empty State component provides consistent messaging when users encounter screens with no content. It guides users toward taking action while maintaining a friendly, encouraging tone.

Purpose

Guide users when no content exists

Locations

Merchants, Wishlist, History, Search results

Icon Circle

64px, bg-warm-100 with warm-400 icon

CTA Button

Primary emerald button for main action

Component Structure

Anatomy

Usage Guidelines

Do

  • • Use friendly, encouraging language
  • • Always provide a clear action (CTA button)
  • • Keep descriptions concise (1-2 sentences)
  • • Use icons that relate to the content type
  • • Center the component vertically in available space

Don't

  • • Use negative or discouraging language
  • • Leave users without a next step
  • • Write long paragraphs of explanation
  • • Use generic or unrelated icons
  • • Place at the very top of a scrollable page
