Profile Dropdown

Context-aware dropdown menu for profile avatar. Shows current role, growth CTAs, and context switching for dual-role users.

Overview

The Profile Dropdown appears when clicking the profile avatar in the top navigation. It adapts based on the user's profile configuration:

  • Shopper Only: Shows shopping context + CTA to add professional profile
  • Professional Only: Shows work context (with merchant count if multiple) + CTA to add shopping profile
  • Both Profiles: Shows current context + switch option (no growth CTA)

Behavior Notes

Growth CTA Removal

The "Use Saltwyk for Business" / "Shop with Points" CTA only appears when the user has a single profile type. Once they activate both profiles, the growth CTA is replaced by the context switch UI.

Context Switch Behavior

Clicking the switch card triggers an instant context change. No modal, no loading spinner. The sidebar theme transitions (light â†” dark), URL updates, and content loads seamlessly.

Handle-Based Identity

Unclaimed: Auto-generated as @firstname_lastname_4digits (e.g., @john_doe_4827)
Claimed: User chooses custom handle (e.g., @johndoe) with full name shown below
Claim CTA: Shows "Claim your custom name" link that opens claiming modal

Sandbox Testing (Merchant Affiliates)

Users with merchant relationships can access Sandbox Testing mode to preview the shopper experience. Sandbox uses purple/indigo accent colors to clearly differentiate from live contexts. Navigates to /passport/test or similar sandboxed route.

Multiple Merchant Handling

Single merchant: Click â†’ Instant context switch (no modal).
Multiple merchants: Click "Work (3 merchants)" â†’ Opens Merchant Picker Modal.
User selects a merchant via radio button, then clicks "Switch Context â†’" to complete the switch. Modal closes on: selection + button click, backdrop click, or ESC key.

Avatar Color

Shopping context: Warm gray avatar (bg-warm-300).
Professional context: Emerald avatar (bg-emerald-600) indicating active work mode.

When a user has multiple merchant relationships, clicking "Work (N merchants)" opens this modal to select which merchant context to switch to.
