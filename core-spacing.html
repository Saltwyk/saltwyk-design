Common Spacing Patterns

Component Internal

  • p-2 — Tight buttons, badges
  • p-3 — Input fields, small cards
  • p-4 — Standard cards, containers
  • p-5 — Comfortable card padding
  • p-6 — Large cards, modal content

Gaps (Flex/Grid)

  • gap-1 — Icon + text pairs
  • gap-2 — Tight button groups
  • gap-3 — Form field rows
  • gap-4 — Card grids, lists
  • gap-6 — Section children

Vertical Rhythm

  • space-y-1 — Tight list items
  • space-y-2 — Form labels to inputs
  • space-y-4 — Paragraphs, cards
  • space-y-6 — Sections within page
  • space-y-12 — Major page sections

Page Layout

  • px-4 sm:px-6 lg:px-8 — Page gutters
  • py-8 — Standard section padding
  • py-12 — Large section padding
  • py-16 lg:py-24 — Hero sections
  • mt-16 mb-8 — Header spacing
