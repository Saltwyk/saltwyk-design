Foundation Component

Tables

Data tables for displaying financial transactions, metrics, and structured information. Designed for scannability and comparison.

01 — Philosophy

Structured Display

Tables display structured data for scanning and comparison across multiple attributes.

Density Balance

Balance information density with readability. Too dense overwhelms; too sparse wastes space.

Progressive Disclosure

Row interactions (hover, select, expand) reveal additional context without cluttering the view.

02 — Anatomy

Container

Header Row

Body Rows

Cells

Footer Row

  • Container: Wrapper with border, shadow, and rounded corners
  • Header row: Column labels with sorting controls
  • Body rows: Data rows with consistent cell formatting
  • Cells: Individual data points with type-appropriate alignment
  • Footer row: Optional totals, pagination, or summary

14 — Responsive Behavior

Horizontal Scroll

Container gets overflow-x-auto. Table maintains minimum column widths. Optional sticky first column.

Priority Columns

Hide low-priority columns at smaller breakpoints. Use hidden md:table-cell. Show expand for hidden data.

Card Transformation

Below breakpoint, rows become stacked cards. Each card shows all data for one row with label-value pairs.

17 — Implementation Checklist

Base Table

  • Container: rounded-lg, border warm-200 (light) / intaglio-700 (dark)
  • Header: warm-50 (light) / intaglio-900 (dark), uppercase, tracking-wide
  • Rows: 48px default height
  • Cell padding: 12px horizontal, 12px vertical

Row States

  • Hover: warm-50 (light) / intaglio-700 (dark)
  • Selected: emerald-50 + emerald-500 left border (light)
  • Selected dark: emerald-500/10 + emerald-500 left border
  • Disabled: 50% opacity, no hover

Column Alignment

  • Text columns: left-aligned
  • Numbers/currency: right-aligned with tabular-nums
  • Actions: right-aligned
  • Status badges: left-aligned or centered

Dark Mode

  • Container: intaglio-800 background
  • Header: intaglio-900 background, warm-400 text
  • Body: warm-100 text
  • Borders: intaglio-700

Sortable

  • Sort icons visible on hover, active on sorted column
  • Cursor pointer on sortable headers
  • Unsorted: ↕, Ascending: ↑, Descending: ↓

Accessibility

  • Proper <th> scope attributes
  • Keyboard navigation for selectable rows
  • ARIA labels for sort state changes
  • Caption or aria-label for table purpose
