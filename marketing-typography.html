Marketing Design System

Typography Treatments

Marketing-specific type treatments extending the product system.

Special Effects

Gradient Text

Use sparingly for key headlines and moments.

Brand Gradient

text-gradient-brand

emerald-500 -> cyan-500. Primary gradient for key headlines.

Thread Gradient

text-gradient-thread

cyan-thread -> magenta-thread. High energy, use very sparingly.

On Dark

text-gradient-brand

Gradients work well on dark backgrounds.

Warm Gradient

text-gradient-warm

intaglio-500 -> intaglio-400. Subtle, sophisticated option.

Combinations

Display + Body Pairings

DM Serif Display

Paired with Inter for body text creates classic editorial contrast. Use DM Serif for all marketing headlines, Inter for everything else.

DM Serif Italic

Use italic for pull quotes, testimonials, and emphasis within serif contexts. Never use for headlines.

Outfit (Brand)

Reserved for logo lockup and brand moments only. Do not use for general headlines or body.

Inter Semibold

For UI elements, buttons, navigation. Also for H3/H4 in product contexts. Clean and functional.

Labels

Serial Number Style

All-caps, tracked out, small. Like currency serial numbers.

Primary Serial

Emerald on light backgrounds. Section labels, category markers.

Color: emerald-500

Secondary Serial

Warm gray for less emphasis. Metadata, timestamps, muted labels.

Color: warm-500

Serial on Dark

Light emerald on dark backgrounds. Same usage as primary.

Color: emerald-300

CSS Definition

.serial {
  font-weight: 600;
  font-size: 11px;
  letter-spacing: 0.12em;
  text-transform: uppercase;
}
