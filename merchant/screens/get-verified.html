Design Decisions

Step Indicator: Completed steps use emerald (success), active step uses indigo (selection), and pending steps use warm-300 (neutral). The active step has a subtle ring to draw attention.

Primary Action: The "Continue" button uses emerald-500 because it represents a positive action (progressing toward verification). This is consistent with the "completed" semantic throughout the flow.
