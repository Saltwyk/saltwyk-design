Design Decisions

Positive Values: Available balance uses emerald-600 to indicate funds ready for withdrawal. This provides immediate visual confirmation of available money, while neutral values (pending, dates) use default ink color.

Progress Gradient: The clearing cycle progress bar uses an emerald-500 → lime-thread gradient to indicate growth/progress. This provides subtle visual interest while maintaining the emerald success association.
