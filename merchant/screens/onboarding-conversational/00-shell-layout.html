2. Top Bar Anatomy

The top bar provides branding, progress indication, and escape hatches. Height: 64px.

Element Token/Style Notes

Border Bottom hsl(var(--intaglio-700)) Subtle separation from content

Progress Dot (Completed) 12px, emerald-500 Finished sections
Get Help Button indigo-600 bg, white text Primary action for assistance

3. Two-Column Layout

Chat column handles the conversation flow. Form panel shows captured data and progress.

Chat Column (60%)

  • Background: hsl(var(--ledger))
  • Right border: 1px solid warm-200
  • Messages area: padding: 32px
  • Max content width: 680px
  • Input area: Sticky at bottom with border-top

Form Panel (40%)

  • Background: hsl(var(--warm-50))
  • Left border: 1px solid warm-200
  • Header: Sticky, 16px font weight 600
  • Content padding: 24px
  • Scrollable: Independent scroll from chat

5. Responsive Behavior

Layout adapts across breakpoints while maintaining the conversational flow.

Breakpoint Chat Column Form Panel Notes
Desktop (>1024px) 60% 40% Side-by-side, both visible
Tablet (768-1024px) 70% 30% Side-by-side, narrower panel
Mobile (<768px) 100% Sheet overlay Floating "View Progress" button triggers slide-up sheet
