Merchant Component

Sidebar Navigation

The merchant dashboard sidebar with status badge, navigation sections, and environment indicators.

1. Overview

The merchant sidebar is the primary navigation for the dashboard. It uses the Fire Opal dark theme (intaglio-900 background) to create clear visual distinction from the light content area and reinforce the "work" context.

The sidebar includes a status badge showing the current environment (Trial, Sandbox, or Live), conditional setup navigation for onboarding, and the main dashboard navigation items.

6. Conditional Rendering

SETUP Section Logic

// Pseudocode

if (merchant.onboarding_complete === false) {

showSetupSection();

showStatusBadge('trial');

} else {

hideSetupSection();

showStatusBadge('live');

}

The SETUP section should be removed from the DOM entirely when onboarding is complete, not just hidden. This prevents empty space and confusion.

Environment Detection

// Pseudocode

switch (environment) {

case 'sandbox':

showStatusBadge('sandbox');

break;

case 'production':

showStatusBadge('live');

break;

default:

showStatusBadge('trial');

}

Sandbox overrides the Trial/Live state. A merchant in sandbox mode always sees the sandbox badge regardless of onboarding status.

8. Design Decisions

Why Dark Theme?

The merchant dashboard uses Fire Opal (dark intaglio) to create clear visual distinction from the shopper experience (Bright Opal). This reinforces that the user is in a "work" context, not a personal shopping context.

Why Status Badge at Top?

Environment awareness is critical for merchants. Placing the badge prominently below the logo ensures users always know whether they're working in Trial, Sandbox, or Live mode before taking any actions.

Why SETUP Before DASHBOARD?

For new merchants, completing setup is the priority. Placing SETUP first guides them through onboarding before they explore dashboard features. Once complete, this section disappears entirely.

Why Indigo Active State?

Indigo provides strong contrast against the dark intaglio background while being distinct from emerald (used for positive states) and orange (used for warnings/trial). The left border creates a clear visual anchor.
