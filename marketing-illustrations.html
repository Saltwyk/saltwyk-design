Foundation

Style Definition

What Makes It "Saltwyk Intaglio"

  • ✓ Currency engraving aesthetic - Fine crosshatching, stippling, parallel line work
  • ✓ Monochromatic sepia tones - Warm browns only, sepia-100 to sepia-700
  • ✓ Ornate decorative elements - Guilloche patterns, scrollwork, rosettes
  • ✓ Allegorical subjects - Scales, hands, coins, flowing forms
  • ✓ Centered, symmetrical compositions - Formal balance

What to Avoid

  • ✗ Photorealistic rendering - Should look engraved
  • ✗ Flat vector style - Need texture, depth, fine detail
  • ✗ Any color besides sepia - No greens, no thread colors
  • ✗ Modern/tech aesthetic - Should feel timeless
  • ✗ Busy compositions - Currency has clear focal points
